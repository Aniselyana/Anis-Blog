Academic Projects & Insights

Augmented Reality

AR IsyaratKu

An educational Augmented Reality application designed to help users learn sign language through interactive 3D experiences. This project integrates Unity and Vuforia SDK to display real-time 3D hand gestures when scanning specialized flashcards, aimed at bridging the communication gap.

3D Animation

3D Room Transformation Animation

A messy room is transformed into a clean, organized, and attractive space using 3D animation techniques. Developed using Autodesk Maya/Blender, this project focuses on texturing, advanced lighting setups, and keyframe animation to deliver a realistic before-and-after interior transition.

Web Development

FluffyCones Website

A business website developed for an ice cream company, focusing on user experience and digital marketing. Built with responsive HTML, CSS, and interactive JavaScript features, this site includes an interactive product menu, customer review slider, and a fully functional contact form.
