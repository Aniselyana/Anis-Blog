# Personal Blog Project - GitHub Portfolio

## Description
This is a personal blog page created as part of the CSD 34203 Special Topics in Software Development course at UniSZA. It serves as a personal project portfolio demonstrating an understanding of the front-end software development process and repository management.

## Features
- **Responsive Design**: Accessible on both desktop and mobile screens.
- **Navigation Bar**: Quick scrolling to Home, About, Blog, and Contact sections.
- **Blog Section**: Contains 2 sample blog articles about software development.

## Technologies Used
- HTML5
- CSS3
